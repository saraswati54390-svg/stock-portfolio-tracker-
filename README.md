Stock Portfolio Tracker

This Python project allows users to track their stock investments. Users can enter any stock name, quantity of shares, and price per share. The program calculates the value of each stock and displays the total portfolio value at the end.

Features:

- Add multiple stocks
- Enter custom stock names
- Calculate individual stock values
- Display portfolio summary
- Calculate total investment value

How to Run:

1. Save the file as "stock_tracker.py".
2. Open it in Python IDLE.
3. Press F5 or click Run → Run Module.
4. Enter stock details when prompted.
5. Type "done" to finish and view the portfolio summary.
